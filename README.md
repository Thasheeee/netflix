#🎬 Netflix Clone

A Netflix Clone web application built with React.js, featuring movie details and trailers powered by the TMDb (The Movie Database) API.

User authentication and session management are handled with Firebase Authentication (Email/Password).

**🚀 Features**

🏠 Netflix-like UI – Browse movies in a Netflix-inspired interface.

🎥 Watch Trailers – Trailers fetched directly from TMDb API.

🔐 User Authentication –

Email/Password signup & login

Secure and persistent sessions with Firebase

**🛠️ Tech Stack**

Frontend: React.js

API: TMDb API

Authentication: Firebase (Email/Password)

Hosting :  Vercel 

**Project Structure**

├── public                 # Images for the main background and the icon
├── src
│   ├── assets             # Images for the application
│   ├── pages              # Home, Login, Player
│   ├── components         # Reusable UI components
├── App.jsx                # Main App file
├── LICENSE                #Apache License 2.0
└── README.md


**Clone The Repo**

git clone https://github.com/Thasheeee/netflix.git
cd netflix

**Install Dependencies**

npm install

**Run the app:** npm run dev
